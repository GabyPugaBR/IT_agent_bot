# Frontend

This is the React frontend for the Constellations IT Support capstone project.

## Environment Variable
Create `frontend/.env` from `frontend/.env.example` if you want to override the backend API URL:

```env
REACT_APP_API_URL=http://127.0.0.1:8000
```

## Commands
```bash
npm install
npm start
npm test -- --watchAll=false
```

## What It Contains
- Mock `Constellations.com` school homepage
- Embedded IT Agent entry points
- Chat UI for knowledge, workflow, escalation, and request submission flows
